# Security Policy

The project team and community take security issues in the Agentic Performance Testing System seriously. We appreciate your efforts to responsibly disclose your findings, and will make every effort to acknowledge your contributions.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities by emailing the project maintainers at **[CONTACT_EMAIL@example.com]**.

**Please do not report security vulnerabilities through public GitHub issues.**

You should receive a response within 48 hours. If the issue is confirmed, we will release a patch as soon as possible depending on complexity.
